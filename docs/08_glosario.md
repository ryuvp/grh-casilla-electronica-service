# 08 - Glosario

Estado: Vigente  
Owner: Equipo funcional + tecnico  
Ultima actualizacion: 2026-03-09  
Proxima revision: 2026-04-09

- Casilla: contenedor logico de mensajes asociado a un titular.
- Titular: entidad duena de la casilla (usuario, dependencia o dependencia externa).
- Bandeja de entrada: mensajes recibidos por el usuario autenticado.
- Bandeja de enviados: mensajes emitidos por el usuario autenticado.
- Adjunto: referencia a archivo almacenado en File Service.
- RemoteAuth: middleware que valida token contra Auth Service.
- PMSG: patron de modelos/stores para CRUD en frontend.
- Backend-driven sorting: ordenamiento resuelto por backend, no por UI.
- Smoke test: prueba rapida de funcionamiento basico.
